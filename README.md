# 👩‍💼 Employee Data Analysis (MySQL)

A data analysis project using MySQL to analyze employee salary trends, performance and department-wise insights.

## Features
- Department wise salary analysis
- Performance distribution
- Experience vs salary insights
- Highest and lowest salary queries
- Department wise max/min salary

## Tech Used
- MySQL
- MySQL Workbench

## How to Run
1. Open MySQL Workbench
2. Open employee_analysis.sql file
3. Run the complete file (Ctrl + Shift + Enter)
4. All tables and data will be created automatically!

## Queries Included
- Basic salary analysis (avg, max, min)
- Department wise analysis
- Performance wise analysis
- Experience wise analysis
- Filtered queries (salary > 50000, etc.)
